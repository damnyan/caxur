# Technology Stack

This project utilizes the following technologies to build a robust and high-performance REST API.

| Category | Technology | Description |
| :--- | :--- | :--- |
| **Language** | [Rust](https://www.rust-lang.org/) | Systems programming language focusing on safety and performance. |
| **Web Framework** | [Axum](https://github.com/tokio-rs/axum) | Ergonomic and modular web framework built with Tokio, Hyper, and Tower. |
| **Runtime** | [Tokio](https://tokio.rs/) | Asynchronous runtime for Rust. |
| **Database** | [PostgreSQL](https://www.postgresql.org/) | Open source relational database. |
| **SQL Toolkit** | [SQLx](https://github.com/launchbadge/sqlx) | Async, pure Rust SQL crate featuring compile-time checked queries. |
| **Documentation** | [utoipa](https://github.com/juhaku/utoipa) | Compile-time generated OpenAPI documentation. |
| **Validation** | [validator](https://github.com/Keats/validator) | Struct validation crate. |
